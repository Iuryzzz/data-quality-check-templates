# Data Quality Check Templates
Проект содержит библиотеку шаблонов проверки качества данных. 
Шаблоны предназначены для проверки данных студентов, преподавателей, сотрудников и других объектов информационных систем.
## Структура проекта
- checks/ — отдельные проверки качества данных.
- templates/ — готовые шаблоны проверок.
## Проверки
- Проверка обязательных полей
- Проверка дубликатов
- Проверка уникальности
- Проверка диапазона значений
- Проверка формата Email
- Проверка номера телефона
- Проверка допустимых значений
- Проверка корректности даты
- Проверка ссылочной целостности
- Логическая проверка данных
## Готовые шаблоны
- Students — шаблон проверки данных студентов.
- Teachers — шаблон проверки данных преподавателей.
- Employees — шаблон проверки данных сотрудников.
- University Database — комплексная проверка базы данных университета.

## Назначение

Шаблоны предназначены для описания правил проверки качества данных.

Каждый шаблон представляет собой набор отдельных проверок, которые могут использоваться при автоматическом анализе данных в информационных системах.
## Преимущества

- повторное использование проверок;
- простое добавление новых шаблонов;
- независимость шаблонов от программного кода;
- возможность применения в различных информационных системах.
